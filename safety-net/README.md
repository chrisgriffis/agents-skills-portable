# Safety Net

The paperback behind the streaming movie.

If your AI access disappears -- enterprise seat revoked, API down, session
crashed mid-push -- this folder is what you operate from. Plain prose,
scannable, indexed. No LLM required to read it.

---

## What this is for

- A peekable reference when you have 30 seconds before a reply.
- A continuity dossier so active pushes don't lose context if the amplifier dies.
- Decision-rule shortcuts so common moves don't need re-derivation.
- Provenance pointers back to full state files when you need detail.

## What this is NOT

- Not a prompt to paste into a personal / non-enterprise LLM. If your state
  contains employer-internal information (names, political reads, strategy
  details), pasting it into a non-boundary tool is a data-handling violation.
  Create a boundary rules document that specifies what CAN and CANNOT go into
  personal AI tools.
- Not a full replacement for live state files. Those remain source of truth.
  This is the compressed, human-readable mirror.
- Not a substitute for your inner-circle proxy. If you lose the amplifier, you
  lean harder on your human allies, not on this folder.

---

## Recommended file structure

Adapt these to your situation. Each file is a scannable card, not a full doc.

| File | When to open it |
|------|-----------------|
| `00-if-access-dies.md` | RIGHT NOW if access just died. Emergency card. |
| `01-people.md` | "Who is X again, and how do I handle them?" One card per player. |
| `02-arcs-live.md` | "What is going on and what is the next move?" |
| `03-week-of-[date].md` | "What ships when this week?" Day-by-day. |
| `04-hard-rules.md` | "What's the rule about X?" Spelling, format, comms discipline. |
| `05-drafting-playbook.md` | "How do I draft to [person]?" Per-recipient register rules. |
| `06-decisions-already-made.md` | "Did I decide that? Yes. Don't reopen." |
| `07-without-the-amplifier.md` | "The amplifier is gone, now what?" Degraded-mode discipline. |

## The emergency card (00-if-access-dies.md) template

Structure your emergency card around these sections:

1. **First five minutes.** Breathe. The active push does not require the
   amplifier. It requires YOU. The amplifier made you faster, not possible.
   Do not announce the access loss to anyone who doesn't need to know.

2. **Boundary rules.** If you can use personal AI tools, what CAN and CANNOT
   go in. Names, project names, initiative names, quoted internal text = NO.
   Generic strategy frames, anonymized drafting help = YES. If you find
   yourself wanting to paste a real message: stop, re-anonymize, then paste.

3. **What's on the clock.** Top 3-5 deliverables with dates. Nothing else.

4. **What you already shipped.** Don't re-do these. They're out the door.
   Bulleted list with dates.

5. **Things that must not break.** 3-5 non-negotiable rules that apply even
   (especially) under pressure. Spelling, format, comms discipline.

6. **If you're about to send something high-stakes.** Who is your calibration
   proxy without the AI? Name them. Describe the channel.

7. **Don't do these.** 3-5 failure modes that get worse under pressure.
   Apologizing for the loss, catastrophizing in writing, piling asks on
   bandwidth-constrained allies, shipping without the format pass.

## Refresh discipline

Stale safety net is worse than no safety net. Refresh triggers:

- Any time a live arc shifts materially (new actor, position change, ship event)
- Weekly at minimum during active pushes
- Immediately after a strategic shift (new adversary, new ally, new deadline)

If the gap between safety net and live state exceeds one week during an active
push, treat the live state as truth and rebuild the affected card.

## How to use it

- **Peek mode (default):** Open the relevant file, scan for one heading, close.
- **Pre-meeting:** Open `01-people.md` (relevant card) + `02-arcs-live.md`
  (the arc the meeting touches). 60 seconds max.
- **Pre-draft:** Open `05-drafting-playbook.md` for recipient +
  `04-hard-rules.md` if outbound to a principal.
- **Lost the thread entirely:** Open `00-if-access-dies.md` then
  `03-week-of-[date].md`.

---

## Portability note

This template is domain-agnostic. Replace the file names and content with
your own situation. The structure -- emergency card, people cards, live arcs,
weekly clock, hard rules, drafting playbook, closed decisions, degraded mode --
is universal for any strategic operator who uses AI as an amplifier and needs
insurance against losing it.
